# 워밍업 예제 Colab 자습 노트북 생성 프롬프트 (먹구름→비, 전체 필사형)

[목표]
`example/` 의 먹구름→비 미니 트랜스포머를 학생이 코드를 직접 타이핑(전체 필사)하며 자습하는 자기완결형 단일 `.ipynb` 노트북으로 재구성.

[역할]
당신은 파이토치 트랜스포머 입문 교보재를 다수 제작한 10년차 ML 강사 겸 Colab 콘텐츠 개발자입니다.

[맥락]
- 내 상황: Colab 번역 과제 전 워밍업 예제를, 학생이 셀마다 코드를 직접 쳐 보며 손으로 익히는 자습 노트북으로 만들고자 함.
- 결과물 독자: 파이썬 기초만 아는 트랜스포머 입문 교육생 (강사 개입 없이 혼자 타이핑·실행).

[입력]
- 모델 소스: `example/mini_transformer.py` (필사 대상 원본)
- 데모 소스: `example/demo.py` (필사 대상 원본)
- 설명서: `example/README.md` (개념 설명 재료)

[처리]
- 타이핑 3종 세트를 단위마다 반복 배치:
  - (a) 마크다운 셀: 이 단위가 하는 일 설명 + 입력할 코드(읽기용 블록, 각 줄에 인라인 주석)
  - (b) 빈 코드 셀: `# 👆 위 코드를 직접 입력하고 ▶ 실행하세요` — 학생이 타이핑
  - (c) (핵심 단위만) 셀 아래 self-check `assert` 로 올바르게 입력했는지 즉시 확인
- 셀 단위 원칙: 실행 가능한 한 단위 = 한 셀. 단문은 한 줄, `def`·`class` 는 통째로 한 셀(여러 줄 블록은 쪼갤 수 없음).
- 전체 필사 범위: 모델(위치 인코딩·어텐션·멀티헤드·인코더/디코더·마스크 각 클래스/함수) + 데모(데이터·사전·학습·생성·히트맵) 전 단위.
- 진행 순서: (0)환경·한글 폰트 → (1)데이터 → (2)토큰화·사전 → (3)모델 정의 → (4)학습 → (5)순차 생성(마스킹) → (6)어텐션 히트맵 → (7)대조 → (8)정리.
- 자기완결형: 외부 `.py` import 금지, 모델 코드 노트북 셀에 인라인. Colab 기본 `torch`·`matplotlib` 사용(추가 pip 불필요).
- 한글 렌더: 나눔폰트 설치(`apt`)·폰트 설정 + `matplotlib.mathtext` 경고 억제, 히트맵은 인라인 출력(`plt.show()`).
- 라인별 설명: 읽기용 코드 블록의 각 줄에 `# 설명` 인라인 주석 필수.
- 출력파일: `example.ipynb`
- 톤앤매너: 한국어, 비유·이모지 활용, 입문자 눈높이.
- 작성 규칙:
  - `nbformat` 4 규격, 코드 셀 `execution_count=None`·`outputs=[]`
  - 고정 시드로 재현성 확보
  - 상단에 Open-in-Colab 안내 문구 포함

[출력]
- `start-here/hands-on/voc-translate/example/example.ipynb` (학생 타이핑용 자기완결형 노트북)

[제약조건]
- MUST:
  - 각 타이핑 단위마다 (설명+읽기용 코드) 마크다운 셀과 (빈) 입력 코드 셀을 쌍으로 제공
  - 학생이 마크다운의 코드를 그대로 입력하면 위→아래로 오류 없이 학습→생성→히트맵 인라인 출력
  - 핵심 단위에는 self-check `assert` 로 올바른 입력 여부를 즉시 확인 가능
  - 크로스 어텐션 히트맵이 질문의 '먹구름이'에 집중함을 확인 가능
- MUST NOT:
  - 빈 입력 코드 셀에 정답 코드를 미리 채워 넣지 않음(정답은 마크다운에만 읽기용으로 표시)
  - 로컬 절대경로·`example/` 파일 import 등 환경 의존 코드를 포함하지 않음
  - GPU를 필수화하지 않음 (CPU에서 수 초 실행)
- 완료조건:
  - `.ipynb` 생성 + 마크다운의 코드를 코드 셀에 채워 실행 시 오류 없이 완주하며 "비가 올 것 같아" 답과 히트맵이 나옴(로컬 venv로 검증)
- 비고: 이 예제엔 빈칸이 없어 '정답 비공개' 제약은 해당 없음. 타이핑할 코드는 필사 학습 특성상 마크다운에 공개.

[예시]
````
[마크다운]  ### 🎯 어텐션 = softmax(QKᵀ/√dₖ)·V
            아래 코드를 직접 입력하세요:
            ```python
            scores = torch.matmul(q, k.transpose(-2, -1)) / math.sqrt(d_k)  # 관련도 점수
            attn = torch.softmax(scores, dim=-1)                            # 합=1 비율로
            context = torch.matmul(attn, v)                                 # V를 그 비율로 가중합
            ```
[빈 코드셀]  # 👆 위 코드를 직접 입력하고 ▶ 실행하세요
[코드셀]     assert context.shape == (1, 3, 8)   # ✅ 잘 입력했어요!
````
